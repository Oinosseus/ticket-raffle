§ load()

load ( )

Load the current settings from the database.

If $db was not set in the __construct() this will ignore the load (no error is returned). If the database row id is unknown (not set in the constructor and raffle not saved) this will also do nothing.

§ save()

save ( )

Saving the actual values to the database.

If no $db was given at __construct() this operation will fail. If the database row id is already set (!=0) this operation will update the database row. If the database row id is not set (==0) this will create a new database row and update the id.

Returns
bool True if raffle was saved successfully.
